API Reference

GET get-feed

Returns a list of recent posts.

  • limit (query): Max posts (def: 20)
POST create-post

Creates a new post on the timeline.

  • content (body): Text content
  • type (body): 'thought', 'project', etc.
GET get-profile

Get public profile data for a user.

  • username (query): Target user
GET search-repos

Search public repositories.

  • q (query): Search term
